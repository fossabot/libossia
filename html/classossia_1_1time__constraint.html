Detailed Description

The time_constraint class.

#time_constraint is use to describe temporal structure precising which amount of time (the duration) are between two #time_events.

The duration can be fixed or between a minimal and a maximal duration.
#time_constraint is also a #Clock and a #TimeProcess container.
